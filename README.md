## About Visit USA

Visit USA is a travel planning application that helps users create personalized travel itineraries for destinations across the United States. The app provides destination information, trip planning tools, maps, and itinerary recommendations.

This application is intended for users aged 18 years and older.

---

## Information We Collect

Depending on how you use the application, we may collect:

### Travel Preferences

* Selected destination cities
* Trip duration
* Budget preferences
* Travel interests and itinerary selections

### Location Information

If you grant permission, Visit USA may access your device location to:

* Display nearby attractions
* Improve travel recommendations
* Provide map-related functionality

Location access is optional and can be disabled through your device settings.

### Device Information

The app may automatically receive limited technical information such as:

* Device type
* Operating system version
* App version
* Crash and performance information
